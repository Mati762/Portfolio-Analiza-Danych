# 📊 Portfolio - Analiza Danych
Cześć! To jest moje repozytorium, w którym publikuję swoje projekty związane z analizą danych, wykorzystując SQL, Pythona i narzędzia BI.

---

## 📁 Projekt 1: Analiza Zarobków w Branży IT (Polska)

### 🎯 Cel projektu
Głównym celem tego mini-projektu była eksploracja danych dotyczących wynagrodzeń na różnych stanowiskach w polskim sektorze IT, z naciskiem na stanowiska analityczne (Data Analyst, Data Scientist). Chciałem sprawdzić m.in. jak język programowania i miasto wpływają na pensję.

### 🛠️ Wykorzystane narzędzia
* **SQL (SQLite):** Filtrowanie, sortowanie i agregacja danych (funkcje średniej, grupowanie).
* **DB Browser for SQLite:** Lokalne środowisko pracy z bazą.
* **Git / GitHub:** Kontrola wersji i publikacja kodu z poziomu konsoli Git Bash.

### 🔍 Zadane pytania biznesowe
W trakcie analizy odpowiedziałem za pomocą zapytań SQL na następujące pytania:
1. Kto w branży IT zarabia powyżej 10 000 PLN?
2. Jak wyglądają najwyższe zarobki dla analityków z mojego regionu (Olsztyn)?
3. Jaka jest średnia pensja na stanowiskach wymagających znajomości języka Python?

### 💡 Główne wnioski z danych
* Stanowiska Seniorskie (np. Data Scientist) mocno zawyżają średnią, oferując zarobki rzędu 22 000 PLN.
* Znajomość języka Python jest mocno opłacalna – średnia pensja dla specjalistów Pythona w tym zestawieniu wyniosła **17 333 PLN**.
